drupal-chef-shared-secret
=========================

Drupal module that provides entity for defining Chef shared secrets.

This module was written to be used with the chef_encrypted_databag module (included with [chef_databag](https://github.com/benclark/drupal-chef-databag) module package), but is a standalone entity definition for future development/expansion.

### Requirements ###

* [Drupal](http://drupal.org) 7.x
* [Entity API](https://drupal.org/project/entity)

### How to use ###

1. Enable module and navigate to `admin/structure/chef_shared_secret`.
2. Create chef_shared_secret entities.
